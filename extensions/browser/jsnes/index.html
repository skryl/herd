Shared Browser Extension

JSNES

Load a local .nes cartridge or an API-provided ROM payload. Caller tiles can claim player 1 or player 2 and drive the shared emulator through the extension API.

Status Choose an NES ROM to begin.
ROM No ROM loaded.
Viewport Keyboard works locally. API control is caller-owned.

Local keyboard: P1 uses arrows, X, Z, Enter, Right Ctrl. P2 uses numpad controls from upstream jsnes.
